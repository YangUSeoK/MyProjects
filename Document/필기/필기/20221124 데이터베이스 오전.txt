Apache24 폴더를 C드라이브에 넣기

Apache24 폴더 중 conf랑 htdocs 폴더가 중요하다.
htdocs는 IP를 타고 들어왔을 때 공개되는 폴더임

conf 폴더에 httpd.conf 메모장으로 열어서
서버위치, 포트, 웹문서 저장위치, 서버네임 설정

37번째 줄에 Define SRVROOT = 설치된 위치
60번째 줄 Listen = 포트
261번째 줄 DocumentRoot = 웹 문서 저장위치
227번째 줄 ServerName = 서버네임. localhost:80 으로 수정 (로컬로 할거임)

설정 다 하고나면
시스템-고급시스템 설정 -> 고급 -> 아래쪽 환경변수
아래쪽 시스템 변수에 Path 더블클릭
C:\Apache24\bin 주소 추가

cmd 관리자권한으로 실행
httpd -k install  엔터
service is successfully installed 문구 확인 
httpd -k start 엔터

인터넷에 localhost 들어가보면 It works! 나오면 정상
httpd.conf 나 php.ini 수정하면 cmd에서 httpd -k restart 해줘야함
